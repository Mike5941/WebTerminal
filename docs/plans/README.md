# Plans

Plans are first-class artifacts when work spans multiple steps, touches several
files, or captures decisions that future agents should see.

## States

- Active: work currently in progress.
- Completed: work finished and kept for historical context.
- Deferred: useful work that is intentionally not being done now.

## Template

```md
# Plan: short-name

Status: Active

## Goal

## Acceptance Criteria

## Steps

## Decision Log

## Verification
```
